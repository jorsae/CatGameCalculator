About

Cat Game Calculator is a calculator for the mobile game "Cat Game - The Cats Collector!"

This calculator will calculate how to spend your resources efficently.

You can input the different items you want to craft as well as the quantity. Then you select how long you want to spend crafting those items. The calculator will then show you how many of each material you need, as well as the costs. The shorter the crafting duration the more it will cost as the price increase when mass crafting.
